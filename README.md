## Getting Started

1. **Clone the repository:**

   ```bash
   git clone https://github.com/Harsha-2324/ReactTask_6.git

   ```

2. **Install dependencies:**

   ```bash
   cd Todo task
   npm install

   ```

3. **Start the development server:**

   ```bash
   npm run dev
   
   ```

4. **Open your browser and navigate to** http://localhost:5173/ **to view the app.**
